## 2024-05-27 - Godot Editor Plugin UX
**Learning:** Adding `tooltip_text` and `placeholder_text` greatly improves usability in complex tool interfaces like Godot editor docks, since the physical space for descriptive text is limited. Changing the cursor shape (`mouse_default_cursor_shape = Control.CURSOR_POINTING_HAND`) makes Godot `Button` controls feel more interactive, which is crucial for web/desktop-app expectations but often missed in default Godot UI. Visual separation using `HSeparator` drastically improves scanability when a dock contains multiple dense logical sections (e.g. Connection vs Actions).
**Action:** When working on Godot Editor plugins, actively add tooltips and pointer cursors to all buttons to align with modern desktop application standards. Use `HSeparator` to group logic vertically.
